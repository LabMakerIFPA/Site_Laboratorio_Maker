Repositório do Arduino

O Arduino é uma plataforma de prototipagem eletrônica open source. Na prática, ele é formado por uma placa eletrônica expansível que pode ser utilizada para o desenvolvimento de protótipos, ou seja, para adicionar inteligência em qualquer coisa e até controlá-la remotamente. Ele também conta com uma IDE Integrated Development Environment para o desenvolvimento do software que será inserido na placa para realizar as atividades programadas. A baixo você encontrará documentos e sites para ajudar na programação montagem etc. do Arduino. Site oficial do Arduino aqui!
